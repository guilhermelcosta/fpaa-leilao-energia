## Problema proposto

Uma empresa produtora de energia possui uma quantidade X de energia para vender. A venda será realizada por um leilão: as interessadas darão lances por lotes de K megawatts, oferecendo um valor V por este lote. As interessadas só comprarão um lote do tamanho exato da oferta e a produtora quer maximizar o seu lucro vendendo sua oferta.
